🎨 CSS Lessons - Styling the Web

CSS (Cascading Style Sheets) is used to control the look and feel of your HTML pages. Let's make your websites beautiful!

1. What is CSS?

CSS stands for Cascading Style Sheets. It describes how HTML elements should be displayed on screen.

/* Example CSS */ body {
  background: white;
  color: #333;
}
h1 {
  color: #00cc88;
}

2. Common CSS Properties

  • color — Text color
  • background — Background color/image
  • font-size, font-family — Typography
  • margin, padding — Spacing
  • border, border-radius — Borders & corners
  • display, flex — Layout control

3. How to Connect CSS to HTML

<!-- External CSS -->
<link rel="stylesheet" href="style.css">

<!-- Internal CSS -->
<style>
  h1 { color: green; }
</style>
