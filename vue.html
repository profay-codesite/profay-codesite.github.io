🌿 Vue.js Lessons - The Progressive JavaScript Framework

Vue.js (pronounced "view") is a progressive JavaScript framework for building user interfaces. It's lightweight, easy to learn, and perfect for both small projects and large-scale applications.

1. What is Vue.js?

Vue.js makes it easy to create reactive and composable UI components. It has excellent documentation and a gentle learning curve compared to other frameworks.

// Vue CDN Example
<div id="app">
  {{ message }}
</div>

const { createApp } = Vue;
createApp({
  data() {
    return { message: "Hello from Profay Codesite! 🚀" }
  }
}).mount('#app');

2. Key Vue Concepts

  • Directives — v-if, v-for, v-bind, v-on
  • Reactivity — Automatic UI updates
  • Components — Reusable UI pieces
  • Props & Events — Component communication
  • Composition API — Modern way with setup()
  • Vue Router & Pinia — Routing and state management

3. How to Set Up Vue.js

# Using Vite (Recommended)
npm create vue@latest my-vue-app
cd my-vue-app
npm install
npm run dev
Tip: You can also try Vue instantly on vue.new or Vue Playground

4. Your First Vue Component

<script setup>
import { ref } from 'vue';
const count = ref(0);
</script>

<template>
  <h1 style="color: #00cc88;">Profay Codesite</h1>
  <p>Count: {{ count }}</p>
  <button @click="count++">Increment</button>
</template>
