🟢 Node.js Lessons - Server-Side JavaScript

Node.js is a runtime environment that allows you to run JavaScript on the server side. It's fast, scalable, and perfect for building backends, APIs, and full-stack applications.

1. What is Node.js?

Node.js uses Chrome's V8 JavaScript engine to execute code outside the browser. It’s event-driven and non-blocking, making it excellent for real-time applications.

// Check if Node.js is installed
node --version

// Simple Node.js file (server.js)
console.log("Hello from Profay Codesite Backend! 🚀");

Try It Yourself - Basic Node.js Concepts

Run these examples locally after installing Node.js:

1. Hello World Server

const http = require('http');

const server = http.createServer((req, res) => {
  res.writeHead(200, {'Content-Type': 'text/plain'});
  res.end('Hello from Profay Codesite! 🚀');
});

server.listen(3000, () => {
  console.log('Server running at http://localhost:3000');
});

Save as server.js and run: node server.js

3. How to Install & Setup Node.js

# Download from nodejs.org

# Initialize a new project
npm init -y

# Install packages
npm install express

# Run your app
node app.js

4. Your First Express Server

const express = require('express');
const app = express();

app.get('/', (req, res) => {
  res.send('Welcome to Profay Codesite Backend!');
});

app.listen(3000, () => {
  console.log('Server is running on port 3000');
});
