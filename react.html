⚛️ React Lessons - Build Modern Web Apps

React is a powerful JavaScript library for building fast, interactive user interfaces. Created by Facebook, it's used by companies like Netflix, Airbnb, and Instagram.

1. What is React?

React lets you build complex UIs from small, reusable pieces called Components. It uses a virtual DOM for high performance.

// JSX - JavaScript + HTML
function Welcome() {
  return <h1>Hello from Profay Codesite! 🚀</h1>;
}

2. Key React Concepts

  • JSX — JavaScript Syntax Extension (HTML in JS)
  • Components — Reusable building blocks
  • Props — Passing data between components
  • State — Managing dynamic data (useState)
  • Hooks — Special functions for React features
  • Virtual DOM — Efficient updates

3. How to Set Up React

# Using Vite (Recommended)
npm create vite@latest my-app -- --template react
cd my-app
npm install
npm run dev

4. Your First React Component

import React from 'react';

function App() {
  return (
    <div>
      <h1 style={{color: '#00cc88'}}>

      </h1>
    </div>
  );
